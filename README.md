# Hangman Game

This Hangman game is implemented using HTML, CSS, and JavaScript. The game features a list of words related to automotive and computer components, displayed in uppercase. Players aim to guess the hidden word by clicking on alphabet buttons. The game provides hints for each word and visualizes incorrect guesses with a hangman illustration.

## How to Play

1. Click on the alphabet buttons to guess letters.
2. Correct letters will be revealed in the word display.
3. Incorrect guesses will increment the hangman image.
4. Reach the maximum incorrect guesses to see the game outcome.
5. Play again by clicking the "Play Again" button in the modal.

## Features

- Dynamic word selection from predefined lists.
- Visual representation of incorrect guesses with hangman images.
- Engaging modal displaying victory or defeat with the correct word.
- Responsive design for an optimal gaming experience on various devices.

Feel free to explore, play, and enjoy this Hangman game!
